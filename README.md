# Scientific Journal of Mining and Mineral Engineering (SJMME)

## Overview

The Scientific Journal of Mining and Mineral Engineering (SJMME) is an international peer-reviewed open access journal dedicated to publishing high-quality research in mining, mineral processing, geomechanics, mine safety, rock mechanics, sustainable mining, and emerging technologies.

## Features

- Responsive Website
- Open Access Journal
- Professional Homepage
- Editorial Board
- Current Issue
- Archives
- Author Guidelines
- Publication Ethics
- Contact Information
